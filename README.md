## Dependencies
### Required
* zsh
* oh-my-zsh
* submodule: zsh-autosuggestions

* a powerline font of your choice

### Optional
* iTerm2 (mac)

### Automatic setup:
sh -c "$(curl -fsSL https://raw.githubusercontent.com/johnlnguyen/dotfiles/master/setup.sh"
